# Aurym Runes · 符文密语

> 将你的秘密锻造成金，只有你允许的人才能看见它的本相。

一个优雅、私密的前端加密工具——把任何文字藏进一枚独特的符文图片里。 锻造符文、分享魔法链接、用密码解开，整个过程像一场只有你们两个人知道的仪式。

🔮 **在线体验**：[https://runes.rbbyq.cn](https://runes.rbbyq.cn)

---

## ✨ 是什么

**Aurym Runes** 是一个纯前端网页应用，你可以：

- 输入任意一段文字，设定一个密码，点击“锻造”下载一枚独一无二的 **符文 PNG**  
- 把这枚符文图片发给朋友，对方拖入页面、输入相同的密码，就能看到你的秘密  
- 或者直接分享一个“魔法链接”，朋友点开就能直接解密（只需密码）  
- 四种主题配色（暗金 / 秘银 / 猩红 / 深海），每次锻造都伴随粒子星火动画  

它把 AES-GCM 加密后的数据藏进了 PNG 的元数据里，图片本身没有任何肉眼可见的异常，看起来只是一张漂亮的几何图案。

---

## 🎯 为什么做这个

不是为了“有用”，而是为了“有趣”。  
这是一份数字时代的私密仪式感：在信息泛滥的世界里，用一枚只属于你们的符文传递一句只有你们能读懂的话。

---

## 🚀 如何使用

### 1. 锻造符文
- 打开页面，在“锻造”面板输入你的秘密文字（支持任意长度）
- 设定一个密码（留空则无需密码）
- 点击 **✨ 锻造并下载符文** → 自动下载一张 `AurymRune_xxx.png`
- 下载按钮下方会出现 **复制魔法链接**，也可直接将图片发给对方

### 2. 解码秘密
- 切换到“文件解码”面板
- 拖入符文图片（或点击选择），输入锻造时设定的密码
- 点击 **🔓 解码秘密** → 屏幕上显示原文

### 3. 魔法链接（更快）
- 锻造后点击 **复制魔法链接**，把链接发给朋友
- 对方打开链接即自动进入解码界面，只需输入密码即可看到秘密

## 🛠️ 技术栈

- 纯原生 HTML / CSS / JavaScript  
- [Web Crypto API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Crypto_API) – AES-GCM 加密  
- Canvas 2D – 程序化生成符文图形  
- PNG tEXt 数据块 – 隐藏密文  
- 零依赖，单个 HTML 文件即可运行

## 📦 部署

由于所有逻辑都在客户端，你可以直接：

1. 下载 `index.html`
2. 用浏览器打开（或部署到任何静态托管服务）
   - [GitHub Pages](https://pages.github.com/)
   - [Netlify](https://www.netlify.com/)
   - [Vercel](https://vercel.com/)
3. 发给朋友一个链接，立刻开始玩

> ⚠️ **重要：本地运行需使用 HTTPS**  
> 本项目使用了 [Web Crypto API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Crypto_API)，出于安全限制，浏览器仅在安全上下文（`https://` 或 `localhost`）中允许使用加密功能。  
> 如果你尝试通过 `http://` 打开本地文件或部署，锻造和解码功能将报错。  
> - 开发时可使用 `http://localhost`（浏览器视其为安全上下文）  
> - 部署到线上时请务必使用 HTTPS（如 GitHub Pages、Netlify 等默认提供）

## 🧪 安全与限制

- 所有加解密均在浏览器本地完成，你的密码和明文不会离开你的设备
- 加密强度依赖 **AES-256-GCM**，是目前公认的安全算法
- 魔法链接的 `#share=` 部分包含加密后的数据（无法直接阅读原文），但仍建议在可信任的渠道分享
- 符文图片本身是标准 PNG，可在任何平台传输而不丢失隐藏的数据（但某些社交平台可能会压缩图片，导致元数据丢失，建议通过文件或魔法链接分享）
